>使用react-transition-group做动画效果

## 安装
```
yarn add react-transition-group
```

## 使用
包含三个组件`Transition`,`CSSTransition`,`TransitonGroup`
``` javascript
import {Transition,CSSTransition,TransitonGroup} from 'react-transition-group';
```
